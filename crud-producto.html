🎯 Use Cases
CrearProductoUseCase.cs
…/UseCases/
Lógica para crear un producto
🎯 CrearProductoUseCase.cs Pasos que ejecuta:
1. Valida que CodigoSap no exista ya
2. Crea instancia de Producto(…)
3. Llama EstablecerTenant y EstablecerCreador
4. Llama repo.AddAsync(entity)
5. Devuelve Result<int>.Success(id)

Usa primary constructor: CrearProductoUseCase(IProductoRepository repo)
ListarProductoUseCase.cs
…/UseCases/
Obtiene lista paginada/filtrada
🎯 ListarProductoUseCase.cs Llama repo.GetAllAsync(tenantId), mapea entidades a ProductoDto y devuelve Result<List<ProductoDto>>.
+ Editar / Desactivar / Eliminar
…/UseCases/
Un UseCase por cada acción de negocio
Regla del patrón:
• EditarProductoUseCase → PUT completo
• DesactivarProductoUseCase → PATCH /estado, llama CambiarEstado()
• EliminarProductoUseCase → DELETE, llama MarcarEliminado()

Desactivar = sigue visible en lista (inactivo)
Eliminar = desaparece de la lista (soft delete)
